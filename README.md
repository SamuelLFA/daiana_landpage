# Daiana Rocha — Landing Page

Static single-page site for psychologist Daiana Rocha. No build tools, frameworks, or package managers — plain HTML, CSS, and vanilla JavaScript.

## Structure

```
index.html   — Single-page layout (Nav, Hero, About, How It Works, Quote, Qualifications, Contact, Footer)
style.css    — All styles; uses CSS custom properties for the color palette
main.js      — Scroll-based nav styling + IntersectionObserver reveal animations
assets/      — me.jpeg (photo), favicon.svg
```

## Running locally

Open `index.html` directly in a browser, or serve with any static file server:

```bash
python3 -m http.server 8080
```
